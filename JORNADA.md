# Minha Jornada com IA e Programação no Celular

## Por que isso?
Sou engenheiro CNC e estou migrando para o mundo do desenvolvimento de software. Queria provar que é possível programar de qualquer lugar, usando apenas o celular, e com a ajuda de inteligência artificial para acelerar o aprendizado.

## Ferramentas que usei
- **Termux**: terminal Linux no Android.
- **GitHub Codespaces**: ambiente de desenvolvimento na nuvem.
- **Aider**: assistente de programação que integra Claude.
- **Claude (Anthropic)**: modelo de IA que gera e edita código.
- **Python, Git, Rust**: linguagens e ferramentas essenciais.

## Desafios enfrentados
- Instalar `aider-chat` no Termux com Python 3.13 exigiu compilar `numpy` e `tiktoken`, o que foi um grande obstáculo.
- Aprendi sobre ambientes virtuais, `--system-site-packages`, e a importância de usar versões de Python compatíveis.
- Descobri que o GitHub Codespaces resolve esses problemas de build em minutos, permitindo focar no que importa: o código.

## O que consegui
- Criei um README.md bilingue (português/inglês) para meu projeto `app-cnc-pages`.
- Adicionei uma função `calibrar_eixos` em Python, com comentários em português.
- Documentei toda esta jornada neste arquivo.

## Aprendizados
- A IA não substitui o programador, mas é uma parceira incrível para gerar código, documentação e até mesmo conteúdo para LinkedIn.
- Programar no celular é viável e pode ser muito produtivo com as ferramentas certas.
- A comunidade open source (Termux, aider, Anthropic) possibilita inovar mesmo com hardware limitado.

## Planos futuros
- Continuar desenvolvendo o `app-cnc-pages` com novas funcionalidades.
- Explorar outros modelos de IA e integrá-los ao meu fluxo de trabalho.
- Compartilhar mais conhecimento sobre programação assistida por IA.
